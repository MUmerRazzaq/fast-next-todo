# Vercel Deployment Examples

This directory contains practical examples for different Vercel deployment scenarios.

## Examples

| Example | Description | Difficulty |
|---------|-------------|------------|
| [nextjs-fullstack](./nextjs-fullstack/) | Next.js with API routes | Easy |
| [react-python-monorepo](./react-python-monorepo/) | React frontend + FastAPI backend | Medium |
| [static-site](./static-site/) | Simple HTML/CSS/JS site | Easy |
| [python-api](./python-api/) | Standalone FastAPI API | Easy |
| [edge-functions](./edge-functions/) | Edge Runtime examples | Advanced |
| [turborepo-fullstack](./turborepo-fullstack/) | Turborepo with multiple apps | Advanced |

## Quick Start

1. Copy the example directory structure
2. Update environment variables
3. Run `vercel` to deploy

## Edge Case Scenarios

Each example includes:
- Basic setup
- Common edge cases
- Troubleshooting tips
